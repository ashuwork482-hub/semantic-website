What is HTML?

HTML ka full form HyperText Markup Language hai. HTML web pages ka basic structure banane ke liye use hoti hai.

HTML ki madad se hum headings, paragraphs, images, links aur forms web page par show kar sakte hain.

HTML koi programming language nahi hai, balkay ek markup language hai jo browser ko batati hai ke content kaise display karna hai.
